# Objetivo

Esse projeto tem como objetivo realizar testes simples de análise de dados usando a biblioteca `polars` do Python, confirmando as fórmulas e conceitos básicos que já estou habituado a aplicar no R.

# Justificativas

Esse movimento faz parte da minha transição do R para o Python, visando me especializar em uma nova linguagem com a mesma proficiência que já tenho em R.

Optei pelo `polars` ao invés do `pandas` pela familiaridade e preferência que tenho com pipelines e encadeamentos de códigos.

# Apoio
